HOW TO INSTALL:
- Siapkan hosting dan domain.
- Buat database pada phpmyadmin.
- Ekstrak Zip2an kedalam File Manager Hosting / FTP
- Akses DOMAINKAMU/install
- Isi form sesuai database yang kamu buat di phpmyadmin tadi
- Klik Next > silahkan isi data pribadi serta data login
- Klik Next > Apabila installasi tertulis BERHASIL, silahkan hapus folder /INSTALL/

MASUK ADMIN PANEL:
- Akses DOMAINKAMU/apanel

CARA UPDATE:
- Ekstrak Zip2an kedalam File Manager Hosting / FTP
  ( Bisa kamu Replace / Hapus data lama dan Ekstrak ulang )
- Masuk phpmyadmin
- Pilih User database yang kamu gunakan sebelumnya
- Hapus tabel setting
- Kembali ke DOMAINKAMU/install
- Kemudian Install seperti biasa
